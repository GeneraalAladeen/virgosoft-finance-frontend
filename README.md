# pimono-finance-app-replica

This is the replica of papara finance app built in vue3. It replicates a mobile app so ideally should be viewed on with mobile view. 


## Project Setup

Make sure the latest version of node is available on system. i.e (node >=22.12.0). To install Node visit `https://nodejs.org/en/download`. 

Clone the repository

```sh
git git@github.com:GeneraalAladeen/virgosoft-finance-frontend.git
```

Switch to repo folder

```sh
cd virgosoft-finance-frontend
```

Install dependencies

```sh
yarn
```

### To Start application

```sh
yarn dev
```
